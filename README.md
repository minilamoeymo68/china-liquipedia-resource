# china-liquipedia-resource

本仓库用于归档和发布多个独立 HTML 页面。所有页面均为静态文件，不依赖任何后端服务。

## 项目简介

这是一个资源归档仓库，主要存放与特定领域相关的独立 HTML 页面。页面内容保持原始状态，仅作存档用途。仓库本身不针对任何具体的网站或域名，仅作为页面集合的托管点。

## 目录结构

```
.
├── pages/          # 存放所有 HTML 页面
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下为各个独立页面，每个页面一个文件夹，文件夹名即为页面标识。
- `assets/` 目录存放页面共用的资源文件，按类型或用途组织子目录。

## 页面归档说明

- 所有页面均为静态 HTML，可直接通过浏览器打开。
- 页面内容可能包含外部引用（如字体、样式、脚本等），请确保网络环境允许加载。
- 归档页面保持发布时的原始状态，不会进行修改或更新。
- 页面可能涉及不同时期、不同来源的信息，请自行判断其时效性和准确性。

## 维护说明

- 本仓库不定期更新，新增页面或修复明显错误。
- 不承诺任何更新频率或长期可用性。
- 如发现页面加载异常或内容错误，欢迎提交 Issue 说明情况。
- 不接受涉及版权、隐私或法律风险的页面归档请求。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/china-liquipedia-resource.git
   ```
2. 直接打开 `pages/` 下的 HTML 文件即可浏览。
3. 也可将整个仓库部署到任意静态托管服务，通过 URL 访问页面。
